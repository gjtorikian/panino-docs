              Punycode.js is bundled with Node.js 0.6.2 and above. To use it with other Node.js versions, use npm to install the punycode module first.

              To access Punycode in your code, add require('punycode').

              A UCS-2 implementation of Punycode.

              Methods

                • punycode.ucs2.decode
                  • Array

                Creates an array containing the decimal code points of each Unicode character in the string. While JavaScript uses UCS-2 internally, this function converts a pair of surrogate halves (each of which UCS-2 exposes as separate characters) into a single code point, matching UTF-16.
