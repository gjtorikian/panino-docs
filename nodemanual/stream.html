stream

                  2 - Unstable

                  A stream is an abstract interface implemented by various objects in Node.js. For example, a request to an HTTP server is a stream, as is stdout. Streams are readable, writable, or both. All streams are instances of undefined.

                  For more information, see this article on understanding streams.
